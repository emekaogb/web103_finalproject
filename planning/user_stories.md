# User Stories

Reference the Writing User Stories final project guide in the course portal for more information about how to complete each of the sections below.

## Outline User Roles

- General user (Users)
- Service provider (Providers)
- Administrator (Admins)

## Draft User Stories

1. As a user, I want to search for nearby resources, so that I can find resources quickly.

2. As a user, I want to filter resources by category, so that I can find the specific type of support I need.

3. As a user, I want to view details about a resource, so that I know how and when to access it.

4. As a user, I want to get directions to a resource using a map, so that I can easily navigate there.

5. As a user, I want to bookmark or favorite a resource, so that I can access them later without searching again.

6. As a user, I want to see whether a service is free or low-cost, so that I can choose options I can afford

7. As a provider, I want to manage and update my organization’s listing, so that users receive accurate information.

8. As a provider, I want to post events or special programs, so that the community is aware of these additional opportunities.

9. As an admin, I want to review and approve new updates to a listing, so that the platform maintains accurate information.

10. As an admin, I want to remove outdated or inappropriate content, so that users are not misled.
